# Position

Position is a low-level component that reports the cursor position on each `mousemove` event on the `onChange` callback function. It's used in `Surface` components.

## Examples

The example below continously reads the user's mouse coordinates:

We can conditionally add the Position component to the DOM when the user holds down the mouse (on the `mousedown` event), and stop reading the coordinates on `mouseup`:
